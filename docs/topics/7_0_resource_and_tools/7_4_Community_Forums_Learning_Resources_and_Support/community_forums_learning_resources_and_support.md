### Getting Started Guide

For those new to GenAI development, we recommend the following learning path:

1. Complete the [Foundations of Prompt Engineering](https://explore.skillbuilder.aws/learn/course/external/view/elearning/17763/foundations-of-prompt-engineering){:target="_blank" rel="noopener noreferrer"} course (2-3 hours)
2. Explore [Party Rock](https://partyrock.aws/){:target="_blank" rel="noopener noreferrer"} to get hands-on experience with AI-powered apps (1-2 hours)
3. Take the [Building Generative AI Applications Using Amazon Bedrock](https://catalog.workshops.aws/building-with-amazon-bedrock){:target="_blank" rel="noopener noreferrer"} workshop (4-6 hours)
4. Experiment with sample applications from the [AWS Samples - Generative AI](https://github.com/aws-samples/generative-ai-use-cases){:target="_blank" rel="noopener noreferrer"} repository

## Making it Practical

To maximize the value of these resources:

1. **Set Clear Goals**: Define what you want to achieve with GenAI before diving in.

2. **Follow a Learning Path**: Use the Getting Started Guide as a foundation, then expand based on your goals.

3. **Engage Actively**: Participate in forums and Slack channels. Ask specific questions and share your experiences.

4. **Practice Hands-On**: Use workshops and sample code to apply what you've learned to real problems.

5. **Stay Current**: Regularly check blogs and newsletters for the latest GenAI developments.


6. **Mind the Costs**: Use pricing calculators and follow cost optimization best practices when experimenting.


Mastering GenAI requires consistent learning and practical application. Regularly revisit these resources as your skills grow.
